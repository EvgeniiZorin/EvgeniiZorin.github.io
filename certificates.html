Additional education certificates

There are different resources that I use for CPD (Continuous Professional Development). I like to do online courses from websites such as Coursera, FreeCodeCamp. Additionally, I enjoy readig textbooks on different topics in Data Science, Statistics, and Programming. Below I outline the courses that I took and my thoughts for each.

Data Analysis with Python - freeCodeCamp
This is a simple introductory course from FreeCodeCamp that teaches data cleaning, processing, and visualisation with Python libraries (Numpy, Pandas, Matplotlib).

Scientific Computing with Python - freeCodeCamp
This course improved my understanding of programming in Python and offered some challenging certification projects.

Relational Database - freeCodeCamp
Following the completion of this code, I learned the essentials of relational databases (in this case, PostgreSQL), different ways to connect to the databases, and write scripts in Bash that would interact with PostgreSQL.

Machine Learning Specialisation - DeepLearning.ai, Stanford (Coursera)
This is an amazing comprehensive course that teaches both theoretical and practical aspects of building ML models. I believe that this might be the best course on ML out there and would highly recommend it!

Essential Math for Data Science - Thomas Nield (O'Reily)
While not as rigorous and comprehensive as other books on Data Science, this one is a relatively easy read that is perfect for revising challenging theoretical concepts in Machine Learning.

Grokking Machine Learning - Luis Serrano
This book is an excellent introduction into the world of Machine Learning - it uses very simple examples to explain different ML algorithms and Gradient Boosting. I believe that this is the best book that a person can choose to start their journey into the world of ML.
